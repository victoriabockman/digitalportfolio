My Projects

Mancala Simulation

Class: Introduction to AI

For this project, I worked alone to simulate the game Mancala using Python. The specifics of this project included creating an AI player that uses the algorithm Minimax to pick the best available option for gameplay. The other player picked a random option during their turn. We also explored the use of alpha-beta pruning to optimize the minimax function. Then, simulations were run to test the win/loss ratio between the two players, as I increased the depth of the minimax function.

Graphing the Healthcare Suppply Chain

Class: Software Engineering Project

For this project, I am working with five other people to graph the healthcare supply chain for the company GHX. We are ingesting huge data files with product/supplier information into a knowledge graph, applying AI techniques to suggest new or updated connections within the graph, and visualizing the data and query scripts using Neo4J. We have scripts written in Cypher, as well as Python and Gremlin code for the main software.
